Projects


Grocery sales prediction

This project leverages machine learning techniques in Python to forecast grocery sales based on historical data sourced from Kaggle. By preprocessing and analyzing various factors such as date, product attributes, promotions, and prices, the system trains models like Linear Regression and Random Forest to predict future sales trends accurately. The predictive models are integrated into a user-friendly web interface using HTML and CSS, allowing stakeholders to input relevant parameters and receive real-time sales predictions, facilitating informed decision-making and strategic planning in the grocery retail sector.

Capsule notion with Ensemble learning

This project utilizes ensemble learning techniques in Python to develop a medical prescription recommendation system based on input symptoms. By leveraging data gathered through Google search for symptom descriptions, the system employs ensemble methods like Random Forest and Gradient Boosting to predict the most appropriate medical capsules or treatments for given symptoms. The goal is to provide accurate and personalized medical advice, enhancing efficiency in healthcare decision-making and ensuring optimal patient care.
